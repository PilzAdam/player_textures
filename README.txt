Player Textures Mod for Minetest-c55
====================================

This mod allows players to use different textures. Just create the front and
back textures in the player_textures/textures/ folder like this:
player_<player_name>.png
player_<player_name>_back.png
and the player with the name <player_name> will have this textures.
